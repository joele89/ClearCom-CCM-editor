  • Configure assignments and talk latch behavior on Keysets. Click a blue label to select a key. Click again to deselect.

  • Assign the Base Station to groups so the station can hear announcements made to the group and reply back to the announcer.

  • Configure the headset settings for the the station.

    • Sidetone Gain: Sets the sidetone gain in dB for the panel.
    • Headset Limit: Use this option to prevent the headset audio becoming too loud.
    • Sidetone Control
      • Tracking: The sidetone level will follow the MAIN volume.
      • Non-Tracking: The sidetone level will be fixed at the level specified by the sidetone gain.
      • Disabled: The sidetone will be permanently turned off.
    • Headset Mic Type: The type of headset connected into the headset port on the Base Station. Selecting Automatic will detect the type autonomously.

  • Switch between fiber and RJ45 (copper) connections. This is where you can adjust the sync offsets as well (ADVANCED).
